Le web et les personne mal-voyantes

L'accessibilité des pages HTML au personnes mal/non-voyante

Selon les balises html, les auto-reader peuvent ne pas fonctionner, il faut donc adapter au personne mal-voyante la construction du HTML

Comment faire ?

Les balises comme :

  • span
  • div
  • b
  • hr

Ne sont pas toujours adaptees

Les bonne balises a utiliser pour structurer sont :

  • article
  • header
  • nav
  • footer
  • strong
  • em

Les outils pour construire sa page

Outils pour les balises HTML acessible

Plusieurs outils en ligne permettent d'avoir une liste des balises accessible au liseurs de pages comme :

  • La documentation MDN
  • La féderation des aveugles de France

Outils pour verifier l'accessibilité de sa page HTML

L'outil principal pour faire cette verification est W3C Validator qui verifie votre site sur les bases des normes W3C, ce qui inclu l'accessibilité au liseurs de pages web.
